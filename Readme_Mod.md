# 资源统计 / RescourcesStatistics

## 介绍

这个mod可以定期记录游戏内各项资源(钢,煤,研发点等),并可通过[mod文件夹]'./World_of_Warships/bin/[最大数字]/res_mods/PnFMods/RescourcesStatistics/'中的'DataCharts.exe'来生成折线图来在展示游戏内各项资源的变化。

## 安装方式

1. 下载压缩包并解压;
2. 找到'res_mods'文件夹
3. 将其中的'PnFMods'文件夹和'PnFModsLoader.py'放入'./World_of_Warships/bin/[最大数字]/res_mods'中

## 下载链接

[https://pan.baidu.com/s/1VfzsP6P-FkrNF5HC40kwHg?pwd=RESS] 提取码: RESS

## 使用方式

在安装完mod后,打开游戏 让任意一项资源的数量发生变动, 就会在[mod文件夹]'./World_of_Warships/bin/[最大数字]/res_mods/PnFMods/RescourcesStatistics/'中看到自动生成的'[玩家名称].dat'文件,此时启动mod文件夹中的'DataCharts.exe',输入相应的玩家名称即可生成统计图。

**注意:由于'[玩家名称].dat'中存储了每一次的记录,所以每次游戏更新时需要将mod文件夹中的'[玩家名称].dat'移至新版本对应文件夹下(一般而言,可以直接将该文件夹整个移动至新版本下),否则之前存储的数据会丢失**

## 卸载方式

1. 删除'./World_of_Warships/bin/[最大数字]/res_mods/PnFMods/RescourcesStatistics'文件夹即可。

## Description

This mod can regularly record various resources in the game (such as Steel, Coal, R&D Points, etc.), and can generate line charts through the 'DataCharts_EN.exe' in the [mod folder]'./World_of_Warships/bin/[maxnumber]/res_mods/PnFMods/RescourcesStatistics/' to display changes in resources in the game.

## Installation

1. Download the zip file;

2. Unzip the file and open;  

3. Find the folder called 'res_mods';

4. Copy them into the path " ./World_of_Warships/bin/[maxnumber]/".

## Download Link

[https://drive.google.com/drive/folders/106XPH0mJ9hFt6mwXCj47km3x9uPehapN?usp=sharing]

## Usage

After installing the mod, open the game to change the quantity of any resource. You will see the automatically generated '[Player Name].dat' file in the [mod folder]'./World_of_Warships/bin/[Maximum Number]/reshmods/PnFMods/ResurcesStatistics/'. At this point, start the 'DataCharts_EN.exe' in the mod folder and enter the corresponding player name to generate a statistical chart which will be pressented on browser.

**Note: Due to the storage of every record in '[Player Name].dat', it is necessary to move the '[Player Name].dat' in the mod folder to the corresponding folder of the new version each time the game is updated (usually, the entire folder can be directly moved to the new version), otherwise the previously stored data will be lost**

## Uninstallation

1. Open the " ./World_of_Warships/bin/[maxnumber]/res_mods/PnFMods "

2. Delete the folder called 'RescourcesStatistics'
